# agi.llms：模型接入抽象层

`agi.llms` 屏蔽不同模型/供应商调用差异，对上层任务提供统一接口。

## 模块职责

- 统一封装文本、图像、语音、多模态能力调用。
- 隔离第三方 SDK 细节，降低任务层复杂度。
- 提供工厂化模型创建入口，便于切换模型与参数。

## 关键文件

- `model_factory.py`：模型实例创建与选择。
- `base.py`：通用基类或公共协议。
- `text2image.py` / `image2image.py`：图像生成能力封装。
- `speech2text.py` / `tts.py`：音频能力封装。
- `multimodel.py`：多模态模型调用封装。
- `rerank.py`：重排模型封装。

## 扩展指南

新增模型时建议：

1. 在对应能力文件新增实现类；
2. 在 `model_factory.py` 注册模型键与构造逻辑；
3. 在配置中加入环境变量（URL、模型名、推理参数）；
4. 补充 `tests/llms` 用例。
